# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-19

### Added
- Initial release of Trading AI Agent
- Core trading loop with automated execution
- AI decision engine with OpenAI and Anthropic support
- Technical indicator analysis (RSI, MACD, EMA, Bollinger Bands, ATR)
- Hyperliquid exchange integration
- Backtesting engine with performance metrics
- Comprehensive logging with Winston
- CLI interface with multiple commands
- Risk management features (stop-loss, take-profit, daily limits)
- Dry run mode for safe testing
- Configurable scheduler with trading hours
- Market analysis tools
- Decision reporting and statistics
- Complete test suite with Jest
- TypeScript strict typing throughout
- Detailed documentation and README
- Architecture diagrams
- Contributing guidelines

### Features
- ✅ Real-time market data fetching
- ✅ Multi-indicator technical analysis
- ✅ AI-powered trading decisions (GPT-4/Claude)
- ✅ Automated order execution
- ✅ Historical backtesting
- ✅ Position sizing and risk management
- ✅ Daily trade and loss limits
- ✅ Confidence-based trade filtering
- ✅ Structured logging with rotation
- ✅ Error handling with retry logic
- ✅ Graceful shutdown handling

### Security
- Environment variable configuration
- API key protection
- Live trading safeguards
- Dry run mode by default
